## Проект аналитики на данных с Kaggle.ru

### Цель проекта — оценить эффективность продукта, юнит-экономику. Проект включает:

* загрузку CSV файлов

* первичную проверку данных

* загрузку данных в PostgreSQL

* расчёт бизнес-метрик с помощью SQL

* визуализацию ключевых показателей

Используемые технологии:

* Python

* Pandas

* PostgreSQL

* SQLAlchemy

* SQL

* Matplotlib

* Seaborn

### Примеры бизнес-вопросов

В рамках анализа можно ответить на следующие вопросы:

* сколько пользователей делает покупки каждый месяц

* какие товары приносят наибольшую выручку

* какой средний чек у заказов

* сколько заказов делает один клиент

* какое распределение оценок у отзывов


Файл visualization.py строит графики:

* Активные пользователи ежемесячно

* Топ категорий товаров по продажам

* Распределение оценок отзывов

### Запускаем: 
* pip install -r requirements.txt - установить зависимости
* python sql_loader.py - загружаем данные в БД 
* psql -d olist_db -f metrics.sql - выполняем запросы
* python visualization.py - строим визуализации
